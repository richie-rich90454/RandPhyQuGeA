## [Unreleased]

### Added
- Cross-platform desktop application with Avalonia UI
- Blazor WebAssembly web application
- Physics question generation from specification files
- Multiple choice and short answer question types
- LaTeX rendering via MathJax and local pdflatex
- LRU-cached LaTeX renderer
- Export to PDF (QuestPDF), HTML (MathJax CDN), Markdown, and plain text
- Seeded and uniform random number generators for reproducibility
- NCalc-based expression evaluator with degree-based trigonometry
- Common mistake and random offset distractor generators
- Plain text and LaTeX solution builders
- PartOneTextLoader for parsing specification files
- Help window and About dialog
- Keyboard shortcuts (Ctrl+N, Ctrl+E, F1)
- GitHub Actions CI (Windows, macOS, Linux)
