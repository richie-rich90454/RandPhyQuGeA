## [Unreleased]

### Added
- Flutter + Rust cross-platform rewrite replacing .NET/Avalonia/Blazor
- Rust core library with WASM and native FFI support
- Flutter UI with Material 3 theming
- Specification parser supporting [UNIT], [TOPIC], [SKILL], [TEMPLATE] sections
- Expression evaluator with trigonometric, logarithmic, and hyperbolic functions
- Random variable generation with seeded reproducibility
- Question generation with multiple choice, short answer, and filtering
- Export formats: HTML, Markdown, Text, PDF, JSON, CSV, LaTeX
- Dark/light theme support with Material You
- Focused practice mode with immediate feedback
- Mental practice mode with timer
- Session summary with statistics
- Progress tracking with history
- Question bank browser
- Settings with specification file picker
- Difficulty calibration and adaptive suggestions
- Question cache with LRU eviction
- Formula library for physics reference
- Template analysis and validation
- Specification merger
- Unit conversion utilities
- Streak tracking and achievements
- Daily challenges
- Leaderboard service
- Analytics and haptic feedback
- Data sync import/export
- Share functionality
- Accessibility helpers and semantics
- Adaptive layout for responsive design
- Theme customization
- GitHub Actions CI/CD workflows
- Docker multi-stage build
- Comprehensive test suite (unit, integration, edge case, property, pipeline)
